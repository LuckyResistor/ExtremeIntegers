## How to Build the Examples

```none
> cd ~
> git clone https://github.com/LuckyResistor/ExtremeIntegers.git ExtremeIntegers
> mkdir ExtremeIntegers-build
> cd ExtremeIntegers-build
> cmake ../ExtremeIntegers
> make
> 01-types/01-types
```
